Contact Book Application 📞

A Contact Book application developed using Python and Tkinter that provides a simple graphical user interface for managing personal contacts. Users can add, view, search, update, and delete contact information, with all data stored locally in a text file for persistence.

Features
Add new contacts
View all saved contacts
Search contacts by name
Update existing contact details
Delete contacts
User-friendly graphical interface
Data persistence using text files
Phone number validation
Technologies Used
Python
Tkinter (GUI)
File Handling
How to Run
Install Python on your system.
Download or clone this repository.
Open a terminal in the project folder.
Run the application:
python contact_book.py
Contact Information Stored

Each contact includes:

Name
Phone Number
Email Address
Address
Functionalities
Add Contact

Save new contact details with validation for required fields and phone number format.

View Contacts

Display all saved contacts from the database file.

Search Contact

Find a contact using the person's name.

Update Contact

Modify an existing contact's information.

Delete Contact

Remove a contact from the contact book.

Exit Application

Close the application safely.

Data Storage

Contact records are stored in a local file named:

contact_book.txt

This ensures that contacts remain available even after closing the application.
